2. Important notes

Preparing your reads

Alaska accepts compressed files, as well as plain folders/files. If you decide to compress your reads, please make sure it is in one of the supported formats. .tar.gz is recommended. Just make sure each sample is compressed into separate files (i.e. one file per sample) or in separate folders (i.e. one folder per sample).

3. Raw reads submission

Uploading your reads

You will be uploading your reads via FTP. For large files, you may wish to use a server.

If you aren't familiar with transferring files via FTP, an FTP client, like Cyberduck, may be helpful.
